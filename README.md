# Online Ticket Booking System

Ein Webprojekt zum Online-Buchen von Tickets, entwickelt mit **Python** und **Django**.  
Benutzer können Tickets ansehen, buchen und verwalten. 
Das System bietet eine einfache und benutzerfreundliche Oberfläche für die Ticketreservierung.

## Features

- Benutzerregistrierung und Login
- Tickets anzeigen
- Tickets online buchen
- Buchungshistorie anzeigen
- Admin-Bereich zur Verwaltung von Tickets und Benutzern
- Responsives Design

## Technologien

- Python
- Django
- HTML
- CSS
